Pular links

Início do Conteúdo

Navegação rápida e fácil
A opção para pular links é muito útil para os usuários que navegam com o teclado, com ou sem a ajuda de leitores de tela, inclusive para usuários com pouca destreza manual que não conseguem usar o mouse ou usuários de laptops, notebooks ou celulares. Pular links é fundamental para quem navega via teclado, pois faz a “rolagem” da tela, pulando vários links que seriam lidos, ou textos anteriores ao que se deseja chegar.

Um exemplo de técnica muito usada é adicionar no início da página o seguinte link:

<a href=”#1s”>Salto para Conteúdo.</a>
<a name=”1s”>Título do Conteúdo Principal.</a>
